# SpaceV — Космическая RPG

**Движок:** Godot 4.4  |  **Язык:** GDScript  |  **Жанр:** Space Trading / Combat RPG

---

## О проекте

SpaceV — процедурно генерируемая космическая ролевая игра. Игрок управляет космическим кораблём, торгует между 36 звёздными системами, выполняет задания, сражается с пиратами и строит собственную фракцию.

---

## Ключевые системы

### 1. Главное меню (`scenes/main/Main.gd`)
- Анимированный фон: звёзды, туманности, галактическая спираль, падающие звёзды
- Кнопки: **Новая игра**, **Продолжить** (загрузка сохранения), **Выход**
- Начальный корабль: «Зонд-С» с 500 000 кредитами

### 2. Карта галактики (`scenes/galaxy_map/GalaxyMap.gd`)
- **36 звёздных систем** в 2D-пространстве, связанных 40+ маршрутами
- **Туман войны:** неизвестные системы скрыты, соседние — видны как «???»
- **Перелёт:** стоимость `max(300, расстояние × 2)` кред., топливо `расст. × 0.12`, время `расст. / 130` дней
- **Встречи в гиперпространстве:** шанс `8% + опасность × 5.5%`
- **Нижняя панель навигации:** быстрые кнопки «Войти в систему», «Мой корабль», «Сохранить», счётчик квестов

### 3. Звёздная система (`scenes/star_system/StarSystemView.gd`)
- 5–12 процедурно генерируемых планет вокруг звезды
- **Враги:** 1–4 пирата (Scout / Fighter / Heavy / Dreadnought / Elite Flagship)
  - Патрулируют систему, преследуют игрока, блокируют космопорты
  - После уничтожения всех врагов космопорты открываются
- **Посадка:** подлетите к планете (≤ 48 ед.) → открывается космопорт

### 4. Космопорт (`scenes/spaceport/Spaceport.gd`)
9 вкладок:

| Вкладка | Описание |
|---------|----------|
| 🍺 Бар | Новости, выполнение заданий, управление фракцией |
| 💰 Торговля | Купля-продажа 8 видов товаров |
| ⚔ Оружие | Установка/снятие вооружения (7 типов) |
| 🚀 Корабли | Покупка кораблей (21 вариант, 8 классов) |
| 📋 Задания | Доска квестов (15 типов заданий) |
| 🏦 Банк | Депозит (4%/день), кредиты (12% разово) |
| 🔧 Ремонт | Ремонт корпуса, заправка топлива, боезапас |
| 🔬 Улучшения | 6 активных/пассивных улучшений корабля |
| 🏛 Штаб | Управление своей фракцией (только лидеру) |

### 5. Боевая система (`scenes/combat/CombatScene.gd`)
- Вид сверху, реальное время, арена 1920×1080
- **Управление:** ЛКМ — движение, `1-4` — выбор оружия, `E` — отступить
- **Улучшения в бою:** `Q` Залп, `W` Щиты, `R` Форсаж, `F` Перегрузка, `X` Инжектор

### 6. Мостик корабля (`scenes/ship_view/ShipView.gd`)
- Вкладки: Системы, Характеристики, Груз, Задания, Личное дело
- Просмотр повреждений, активных квестов, статистики боёв

## Система заданий (15 типов)

| Тип | Условие выполнения | Награда |
|-----|-------------------|---------|
| 📦 Доставка груза | Перевезти X единиц товара в систему | 800–2000 |
| 💀 Охота за головой | Победить в 1+ бою (принять в баре) | 1500–4000 |
| 🛡 Эскортирование | Прибыть в систему назначения | 1200–3000 |
| 🆘 Спасательная операция | Прибыть в систему | 600–1800 |
| 🕵 Контрабанда | Перевезти груз в систему | 2000–5000 |
| 🔭 Патрулирование | Прибыть в систему + провести 3 дня | 500–1200 |
| 🗺 Разведка системы | Прибыть в систему и исследовать | 900–2500 |
| ⛏ Добыча ресурсов | Перевезти руду в систему | 700–1800 |
| 🎯 Устранение цели | Победить в 1+ бою (принять в баре) | 3000–7000 |
| 💾 Извлечение данных | Прибыть в систему | 1800–4500 |
| 🤝 Дипломатическая миссия | Прибыть в систему | 1000–2800 |
| 🏰 Оборона базы | Прибыть + победить в бою в этом секторе | 2000–5500 |
| 👁 Исчезновение корабля | Прибыть + победить в бою (расследование) | 1200–3200 |
| ⚔ Наёмник фракции | 3 победы в боях (принять в баре) | 2500–6000 |
| 💰 Торговый рейс | Прибыть в систему назначения | 400–1500 |

**Система проверки условий:** при принятии задания фиксируются `battles_won_at_accept` и `day_at_accept`, что позволяет точно отслеживать прогресс по боевым и временным условиям.

### Создание своей фракции
- **Стоимость:** 50 000 кредитов
- **Выбор штаба:** раскрывающийся список безопасных систем (опасность ≤ 2, без пиратов/пустоши)
- **Безопасные системы для штаба:** Sol Prime, Krath Station, Auren Gate, Nova Reach, Helion Crossing, Pax Harbor, Silk Route, Drift Market, Echo Station, Relay Point, Hyperion Falls
- **Преимущества лидера:** вербовка союзников, ежедневный доход, вкладка Штаб

## Экономика

- **Кредиты:** торговля, задания, доход от союзников
- **Банк:** депозит (мин. 20 000, +4%/день), кредит (+12% разово)
- **Топливо:** 100 ед. максимум, расход `расст. × 0.12`, цена 12 кред./ед.
- **Ремонт:** 5% цены корабля за полный ремонт корпуса
